# hackathon-Bridge
**ETHSilesia Hackathon Project by Yevheniia (Żeniula) Muravska** - SOLO PARTICIPANT

**Concept:** Merging IoT with Web3 to create a seamless, unique user experience and build a network of loyal customers in the event and entertainment industry.
**Usecases:** 

**Tech Stack:**
* **Hardware:** Arduino/C++ for ESP32 NFC programming;
* **Backend:** Python/Flask – the core that makes the "magic" happen, handling real-time communication between the physical hardware and the web software, and parsing data into a temporary JSON database;
* **Frontend:** HTML/CSS/JS – user interface and WalletConnect/Web3 integration (built mostly by me, with structural help from Gemini);
* **Graphics:** Custom assets generated by Gemini.
